## CARA INSTALL DI TERMUX
```bash
> pkg install nodejs && pkg install git
> git clone https://github.com/X7xSec404/SELFBOT
> cd SELFBOT
> bash install.sh
> npm start/node index
```
## CARA INSTALL DI LAPTOP
```bash
> git clone https://github.com/X7xSec404/SELFBOT
> cd SELFBOT
> npm i
> npm start/node index
```
